## 📱 機能

- リアルタイム音声認識（日本語/英語）
- AI医療アシスタント
- 医療サマリー自動生成
- オフラインモード対応

## 🏗️ アーキテクチャ

- **フロントエンド**: iOS (SwiftUI) / Android (Jetpack Compose)
- **バックエンド**: Azure Functions
- **AI**: Azure OpenAI + Speech Services
- **ストレージ**: Azure Blob Storage + Cosmos DB
